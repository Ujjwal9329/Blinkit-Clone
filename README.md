# Blinkit Clone 🛒

A responsive clone of the Blinkit website built using **HTML**, **CSS**, and **JavaScript**.

## 🔥 Features

- 📦 Product Listings
- 🗂️ Category Sections
- 🛒 Add to Cart Functionality
- 📱 Mobile Responsiveness
- ✨ Modern UI Design
- ⚡ Built using **Vite** for faster development
